<br><br>
补充：
1. 若运行项目后遇到异常情况，那么清除缓存可能会有所帮助，即 `Invalidate Cache` 操作。

2. 若遇到依赖文件下载不下来的情况，这是由于大部分依赖文件是放在国外的服务器上的，可能由于网络问题而导致不易下载。遇到这种情况，可以尝试替换资源镜像为阿里云镜像，配置教程可参考：https://www.jianshu.com/p/3b27628add88




另：科学上网才可以解决绝大部分异常情况。  (˶ᵔ ᵕ ᵔ˶)
